# TaskFlow

TaskFlow is a simple team project management web application built for our final capstone project.

It allows managers to assign tasks and deadlines while employees can view tasks and update their progress.

---

## 👩‍💻 Team Members

- Samira — Frontend & UI/UX
- Niyat — Backend & Database

---

## ✨ Features

- JWT Authentication
- Manager & Employee Roles
- CRUD Operations
- Task Assignment
- Progress Tracking
- Responsive Design

---

## 🛠️ Tech Stack

- HTML
- Tailwind CSS
- JavaScript
- Node.js
- Express.js
- MongoDB
